# writing

Claude Code plugin for writing quality analysis and improvement skills.

## Install

```bash
# Register marketplace (one-time)
claude plugin marketplace add howie/yibi-stack

# Install plugin
claude plugin install writing@yibi-stack
```

## What you get

| Component | Description |
|-----------|-------------|
| `detect-ai-slop` skill | 系統化辨識 AI 生成文字，含多層信號交叉比對、模型特徵比對（ChatGPT / Claude / Gemini）與去除 AI 味建議 |
