# tdd

Claude Code plugin for Test-Driven Development methodology.

## Install

```bash
# Register marketplace (one-time)
claude plugin marketplace add howie/yibi-stack

# Install plugin
claude plugin install tdd@yibi-stack
```

## What you get

| Component | Description |
|-----------|-------------|
| `tdd-kentbeck` skill | Kent Beck 原教旨 TDD：Red-Green-Refactor 循環、測試優先、Baby Steps |
| `flutter-tdd` skill | Flutter 特化 TDD：widget/integration test 優先，Bloc/Riverpod 模式 |
| `ci-triage` skill | CI 失敗快速分析與修復流程：log 解析 → 根因分類 → 修法建議 |

> **qa-test-design** skill（測試設計框架方法論）已移至 `sdd@yibi-stack` plugin。
